JavaScript Calculator
A simple JavaScript calculator built with HTML, CSS, and JavaScript.

Project Overview
This project is a basic implementation of a calculator that performs arithmetic operations. It features a user-friendly interface with buttons for numbers, operators, and a clear function.

Features
Performs arithmetic operations (addition, subtraction, multiplication, division)
Clear function to reset the calculator
Delete function to remove the last character entered
User-friendly interface with buttons for numbers and operators
Technologies Used
HTML5
CSS3
JavaScript (ES6+)
File Structure
index.html: The main HTML file that defines the calculator interface
styles.css: The CSS file that styles the calculator interface
index.js: The JavaScript file that contains the calculator logic
How to Use
Clone the repository to your local machine
Open index.html in a web browser to use the calculator
Click on the buttons to enter numbers and operators
Use the clear function to reset the calculator
Use the delete function to remove the last character entered
